# MirAI Assignment One

Python internship assignments completed during the MirAI AI Builder Internship.

## Projects

### app.py
- Google Gemini API integration
- Sends prompts to Gemini
- Receives and prints AI responses in the terminal

### calculator.py
- Basic calculator built with Streamlit
- Supports addition, subtraction, multiplication and division

### identityechointerface.py
- Streamlit form application
- Collects user name and message
- Validates input
- Estimates token usage based on message length

## Technologies

- Python
- Streamlit
- Google Gemini API

## Run

streamlit run app.py
streamlit run calculator.py
streamlit run identityechointerface.py
